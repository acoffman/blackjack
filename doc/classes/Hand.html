Public Class methods

new(cards, from_split)

A hand needs to consist of at least two cards. More can be added later but an array of two cards needs to be passed to the constructor to start. Additionally, the hand needs to know if it is the result of a split because if it is, a 21 cannot result in a blackjack.

Public Instance methods

[](index)

Convenience method so that you can get the card at position i in the hand without accessing the cards array directly

[]=(index, card)

Allows cards in the player‘s hand to be reassigned. Used primarily for splits

add_card(card)

When taking a hit, you will need to add a card to your hand

blackjack?()

If only two cards are dealt, the hand does not result from a split and the cards total up to 21, then we have a blackjack.

bust?()

Is the hand over 21, even its minimum value (if aces)

hand_value()

Returns the numeric value of the hand as a one element array or a multi element array if more than one possible value.

splitable?()

Returns true if the hand is eligible to be split. ie are there only two cards and do they have the same value?

to_s()

Prints out the player‘s hand as a string, and then the numeric value of the hand (or bust if it is)
