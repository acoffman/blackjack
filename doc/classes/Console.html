Module Console
In: src/console.rb

This module contains the IO methods that deal with requesting and recieving information from the users. It processes it and sends it back to the main program in a usable form. This module is also responsible for ensuring the values passed back are valid in the context of the game

Methods

Public Instance methods

add_human_player(player_num,players)

Asks the player for a name or supplies a default value if none is entered and adds the player to the players array at the table

clear_screen()

Clears the console window (unless your console is really really tall!)

display_dealer(dealer, round_over)

Displays the dealers face up card to the screen Or if the round is over, displays the Dealer‘s entire hand

display_hand(hand)

Displays the players hand along with potential values to the screen

display_message(msg)

displays a given message to the screen

display_result(result,bet)

Prints the result of a hand to the console

display_turn(player)

States the current player and their available money using the overidden to_s method in the player class

divider()

Defaults to 55 chars wide, but that could easily be changed if desired

get_bets(players)

Cycles through each player in the game and collects their bets ignores the dealer which has no bet This function also returns an array to the main program to determine if a player wants to quit or not

is_integer?(input)

Used to make sure that a string input is a valid integer

prompt(hand_num,player)

Displays the appropriate prompt for the given hand and returns the result back to the main program If the Player is a Dealer, it generates the response automatically This function also checks to see what moves are valid before allowing the user to select them

set_up_players()

Asks the user how many players would like to play Then gets names, creates the players and adds them to the table
